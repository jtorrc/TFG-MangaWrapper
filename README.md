# TFG – Solución web para el seguimiento y descubrimiento de obras literarias

**Autor:** Juan Torres  
**Universidad:** Universidad de Huelva  
**Nota:** 9.6/10  

## Descripción

Plataforma web que centraliza el historial de lectura de cómics en línea, sincronizando automáticamente el progreso del usuario independientemente del sitio en el que haya leído.

El sistema se compone de tres elementos:
- **Scraper** (Scrapy + Playwright): extracción de contenido desde múltiples fuentes
- **Plataforma web** (Django + PostgreSQL): gestión del catálogo personal y progreso de lectura
- **Extensión de navegador** (Chrome): detección automática del capítulo leído y sincronización con el servidor
